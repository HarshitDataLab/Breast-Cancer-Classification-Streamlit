# 🩺 Breast Cancer Classification

A Machine Learning web application built with **Python**, **Scikit-learn**, and **Streamlit** that predicts whether a breast tumor is **Benign** or **Malignant** based on diagnostic measurements.

---

## 📌 Features

- Interactive Streamlit web interface
- Real-time breast cancer prediction
- Logistic Regression classification model
- User-friendly input fields
- Fast and accurate predictions

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Breast-Cancer-Classification-Streamlit.git
```

Go to the project folder:

```bash
cd Breast-Cancer-Classification-Streamlit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 📊 Machine Learning Model

- Algorithm: Logistic Regression
- Problem Type: Binary Classification
- Dataset: Breast Cancer Wisconsin Dataset
- Framework: Scikit-learn

---

## 🎯 Prediction Output

The model predicts one of the following:

- ✅ Benign Tumor
- ❌ Malignant Tumor
